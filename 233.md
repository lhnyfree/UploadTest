#### DQL:查询表中的记录
 * select * from 表名;

1.语法：
```mysql
select 
    字段列表            -- 查多个字段用逗号分隔
from
    表名列表            --查多个表用逗号分隔
where
    条件列表
group by
    分组字段
having
    分组之后的条件
order by
    排序
limit
    分页限定
```

2.基础查询
多个字段的查询:
```mysql
select 字段名1,字段名2..... from 表名;
注意如果查询所有字段，则可以使用*号来替代字段列表。  --公司里一般不推荐用
```
去除重复:(只有2个结果集的数据完全一样才可以去除，它某个字段一样，就没用)
```mysql
distinct 
```
